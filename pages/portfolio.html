Viewing Party

Viewing Party aims to compare the watched, favorites, and want-to-watch lists of users to compare and recommend films based off of genre preferences or subscriptions.

Takeaways: Through the viewing party, I learned to work with nested data structures, and also improved my understanding of conditional logic and nested loops.

Swap-Meet

Through the Swap Meet Project, I created various classes and instance methods that would be utilzed to allow different vendors swap items in their inventory according to condition or type. Additionally, I was able to learn more about code coverage through tests and the importance of writing thorough tests.

Takeaways: I became more familiar with creating classes and instance methods, as well as the importance of DRYing up code with the usage of helper functions.

Adagrams

Adagrams was my first peer programming project through which we built a computerized version of anagrams, where each user is given a pool of random letters and the words the user inputs will be returned as a score value.

Takeaways: I learned the importance of active communication with my partner and the value of staying open minded when approaching how we want to code our solution. We did a lot of trial and error together, and I was able to grow immensely through it :).

Solar-System

The Solar System API project was our second pair programming project focused on getting more familiar with SQL databases, SQLAlchemy, and Flask.

Takeaways: During this project, I learned the importance of creating RESTful endpoints for CRUD operations.

Task-List

The task-list project focuses on building an API that will also utilize other web-based APIs (like Slack). The task-list API is able to organize tasks that can be created, read, updated, and deleted, with the addtional abbility to sort tasks, mark them as complete, update in slack, and have goals overarching each task.

Takeaways: I learned how to create models and instance models in SQLAlchemy, as well as create and utilize class and instance methods and higher order helper functions. Additionally, I learned how to create a test database and write tests and test data.

Personal-Portfolio-Site

Takeaways: Through this project, I became more acclimated to the different HTML tags, as well as how to establish a framework for website through flexbox and grid.
